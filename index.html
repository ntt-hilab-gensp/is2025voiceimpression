    • Prompt example (Japanese)

    	# Task
	## 声の印象について、次の11個の印象を1から7の連続値で評価したデータがあります。
	## 各印象に対するスコアは，それぞれ平静時のスコアを表す，
	## 目標に示す話し方で話してもらった時のスコアを作ってください
	## 出力は，印象ごとのjson形式にしてください
	
	# 印象
	## 高い(1)–低い(7): 4.68
	## 男性(1)–女性(7): 1.45

    Prompt example (English translated from Japanese)

    	# Task
	## We have data evaluating the impression of a voice based on the following 11 impressions, using a continuous scale from 1 to 7.
	## The score for each impression represents the score in a neutral state.
	## Generate scores for when the speaker speaks in the target speaking style.
	## The output should be in JSON format for each impression.
	
	# Impresseions
	## High(1)–Low(7): 4.68
	## Masculine–Feminine(7): 1.45
	## Clear(1)–Hoarse(7): 4.16
	## Calm(1)–Restless(7): 3.54
	## Powerful(1)–Weak(7): 4.03
	## Youthful(1)–Elderly(7): 3.99
	## Thick(1)–Thin(7): 3.33
	## Tense(1)–Relaxed(7): 4.1
	## Dark(1)–Bright(7): 3.84
	## Cold(1)–Warm(7): 3.88
	## Slow(1)–Fast(7): 4.20

	# Target
	## Sleepy
